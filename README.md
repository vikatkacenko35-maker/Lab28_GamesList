# GamesApi - Список любимых игр

## Название и описание проекта

**GamesApi** — это веб-сервер на ASP.NET Core для управления списком любимых игр. API поддерживает полный CRUD (создание, чтение, обновление, удаление) игр с хранением данных в памяти сервера.

### Функциональность
- Просмотр всех игр
- Просмотр конкретной игры по ID
- Добавление новой игры
- Обновление существующей игры
- Удаление игры

## Инструкция по запуску

```bash
cd GamesApi
dotnet run
```
| Метод | Маршрут | Описание | Статус |
| --- | --- | --- | --- |
| GET | /api/games | Получить все игры | 200 |
| GET | /api/games/{id} | Получить игру по id | 200 / 404 |
| POST | /api/games | Добавить игру | 201 |
| DELETE | /api/games/{id} | Удалить игру | 204 / 404 |
